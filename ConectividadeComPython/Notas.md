# Como executar
<p>Para você executar você deve utilizar os comandos abaixos</p>

## Windows 
```
    git clone https://github.com/joaovtk/proz-course-database 
    cd  proz-course-database\ConectividadeComPython
    pip install -r requirements.txt
    python main.py
```

## Linux
